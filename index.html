Frequently Asked Questions

Everything you need to know about MellowPDF’s privacy, capabilities, and local execution.

Is MellowPDF really safe for sensitive documents?

Absolutely. MellowPDF is designed with a serverless execution flow. Every operation (merging pages, editing text, watermarking, page rotation, and extraction) runs 100% locally in your web browser. Your document files are never transmitted to any external servers, databases, or API relays.

How can I verify that my files are processed locally?

You can verify this easily by disconnecting your internet connection (Wi-Fi/Ethernet) and using the editor completely offline. You will find that all features—including rendering PDF thumbnails, text editing, and PDF compilation—continue to work. You can open the Privacy Verification link in the footer to read more about this test.

Are there any limits on file sizes or page counts?

No, there are no limitations. Since the processing runs locally on your own computer, the speed and page count limits are only restricted by your computer's RAM and CPU performance. It is completely free for personal and commercial projects.

How does the WYSIWYG text editor work?

When you edit a page, MellowPDF uses PDF.js to extract the text coordinates and font styling properties. When you save your modifications, the compiler masks the original text using a solid white background and redraws the new text in its place, using font styles matched to the document's original typography.

Privacy Policy

Last Updated: June 18, 2026

At MellowPDF (a product of Mellowkraft), we take your privacy extremely seriously. We want to be entirely clear: we do not collect, store, transmit, or share any of your files or personal data.

1. 100% Client-Side Processing

All processing, including loading, rendering, reordering, rotating, page deletion, page extraction, text editing, watermarking, and document compilation, occurs locally in your browser's execution memory using standard WebAssembly and client-side JavaScript libraries.

2. Zero Server Transmission

Your document files never leave your computer. There are no server uploads, cloud storage, API relays, or remote processing servers involved in the workflow. You can disconnect your network connection entirely and continue to use the application.
